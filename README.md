# Solosis-Magic-Ball
A magic 8-ball website but is solosis. Made mostly to practice css animations. The script is very simple, it just randomly picks an answer out of the five there is (regardless of what the user types into the box) when
the send button is clicked. Once the question is sent, the button changes its content to "Ask again?" and restets the whole system when clicked.
The art is based on the black and white sprites and I made it on pixilart.com
